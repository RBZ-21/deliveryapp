# Frontend

This repo uses static HTML pages as the live app:

- `index.html` for the admin dashboard
- `landing.html` for the marketing page
- `login.html`, `setup-password.html`, `driver.html`, and `customer-portal.html` for role-specific screens

The `src/` folder is only a branded React scaffold and should be treated as fallback/demo UI unless you decide to rebuild the frontend around it.
